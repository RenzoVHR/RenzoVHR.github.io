Trash Game

This was my first game project using javascript. In this game you have 3 heart. You have to catch all the trash before it fall to the floor. Each time the trash fall to the floor your recent heart will be decrease by 1 heart. If you do not have any heart, the game is over. And also if you catch the poop the game will be consider game over. As time goes by the level will increase and the trash will fall faster.
